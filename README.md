# Journal-App
A digital journaling platform made with React for recording daily experiences.

## Table of contents
* [Technologies](#technologies)
* [Setup](#setup)

## Technologies
Project is created with:
* JavaScript (ES6)
* HTML5
* CSS3
* React
* Vite
* MaterialUI

## Setup
To run this project, install it locally using npm:

```
$ npm install
$ npm run dev
```
